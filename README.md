# Microcontrollers Laboratory 2023–2024
### ECE NTUA — Academic Year 2023/2024

This repository contains the source code for all laboratory exercises completed as part of the **Microcontrollers Laboratory** course at the School of Electrical and Computer Engineering, National Technical University of Athens (ECE NTUA).

The projects are written primarily in **C** and **Assembly** (AVR), targeting microcontroller platforms used throughout the course.

---

## 📁 Repository Structure

| # | Folder | Topic |
|---|--------|-------|
| 1 | `1.Introductory` | Introductory exercises — basic I/O, GPIO configuration |
| 2 | `2.ExtInterrupts` | External interrupts — ISR handling |
| 3 | `3.Timers_ADC` | Timers & Analog-to-Digital Conversion |
| 4 | `4.LCD_screen` | LCD display interfacing |
| 5 | `5.TWI` | TWI / I²C communication protocol |
| 6 | `6.Keyboard` | Matrix keyboard scanning and input handling |
| 7 | `7.TempSensor` | Temperature sensor integration |
| 8 | `8.IoT_app` | IoT application — end-to-end embedded system project |

---

## 🛠️ Languages & Tools

- MPLAB X IDE or equivalent AVR IDE
- Hardware: AVR ATmega328 microcontroller chip

---

## 🚀 Getting Started

1. Clone the repository:
```bash
   git clone https://github.com/savnastou/MicrocontrollersLaboratory2023-2024.git
```
2. Open the desired lab folder in your AVR IDE of choice (e.g., Atmel Studio or VS Code with the AVR extension).
3. Build and flash to the target microcontroller using your programmer.
